# Chinese Zodiac Calculator

A simple React application that calculates your Chinese Zodiac sign based on your birth year.

## Features

- Input your birth year
- Get your Chinese Zodiac sign instantly
- Clean, responsive design using Tailwind CSS

## How to Run

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser and go to `http://localhost:5175`

## Usage

Enter your birth year in the input field and click "Find My Zodiac Sign" to see your zodiac animal.

Note: This uses a simplified calculation based on the Gregorian calendar. Chinese Zodiac is traditionally based on the lunar calendar, but this provides a good approximation.
